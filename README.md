# Python & Machine Learning Portfolio

Welcome to my repository tracking my training journey at **Besant Technologies**. This repository serves as a practical codebase where I push the Python scripts, data processing workflows, and machine learning models built daily during class and lab sessions.

---

## 📚 Course Overview

This comprehensive program bridges the gap between foundational programming and predictive modeling, covering two major pillars:

### 1. Core & Advanced Python
* Object-Oriented Programming (OOP) concepts, data structures, and file handling.
* Data manipulation and analysis using **NumPy** and **Pandas**.
* Data visualization and Exploratory Data Analysis (EDA) using **Matplotlib** and **Seaborn**.

### 2. Machine Learning Application
* Data preprocessing (handling missing data, feature scaling, categorical encoding).
* **Supervised Learning:** Regression models (Linear, Logistic) and Classification (Decision Trees, Random Forests, SVM).
* **Unsupervised Learning:** Clustering techniques (K-Means, Hierarchical) using **Scikit-Learn**.
* Model evaluation metrics (Accuracy, Precision, Recall, R-squared, Confusion Matrices).
